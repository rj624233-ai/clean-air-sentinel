🌍 Clean Air Sentinel

An IoT-based Air Quality Monitoring System built using ESP32, MQ2 Gas Sensor, and DHT11.

This system monitors air quality in real time, displays it locally, and uploads the data to the ThingSpeak cloud for remote monitoring.


📌 Features

Real-time air quality monitoring

Temperature and humidity measurement

OLED display output

LED & buzzer alert system

Cloud data upload using ThingSpeak

Remote monitoring from anywhere
